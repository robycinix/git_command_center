# Roadmap

## Implemented in 0.1

- Cross-platform Textual shell and repository dashboard
- Educational catalog, live search, category and risk filters
- Intent wizard with visible plans and guarded execution
- Commit graph, branch table, unified diff and reflog recovery timeline
- Four-level learning material and disposable local sandbox
- YAML settings, packaging metadata and automated tests
- Provider boundaries for local/remote AI and GitHub

## Planned

1. Side-by-side syntax-aware diff with virtual scrolling.
2. Three-way conflict editor with hunk-level ours/theirs/both actions.
3. Interactive rebase editor and branch protection configuration.
4. Full exercises, persistent progress and randomized quizzes.
5. Opt-in adapters for OpenAI, Ollama, OpenRouter, Claude and Gemma.
6. OAuth-backed GitHub issues, pull requests, releases and Actions status.
7. Signed release pipelines for Windows, Linux and macOS.
8. Benchmarks on repositories with 100,000 files and 10,000 commits.

Provider integrations must remain opt-in, redact secrets, show transmitted data
and preserve the same command-plan safety boundary as local operations.
